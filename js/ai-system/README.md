# AI Auto System (Mode-wise)

This folder keeps additive AI automation modules separated mode-wise.

- `core.js` - Shared risk summary + panel renderer.
- `customer-mode.js` - Customer auto-generated AI panel.
- `driver-mode.js` - Driver auto-generated AI panel.
- `admin-mode.js` - Admin auto-generated AI panel.

These scripts are wired without removing existing portal logic.
